get /customers
returns all registered customers (customersGet)
Returns all registered customers with pagination

Query parameters

page (optional)
Query Parameter — page index default: 0 format: int32
pageSize (optional)
Query Parameter — page size default: 5 format: int32

Responses

200

returns a page of customers registered

204

no content

400

bad input parameter
Up 
post /customers
register a new customer (customersPost)
Register a new customer to the system

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body customer (required)
Body Parameter — Customer to register

Responses

201

customer registered

400

invalid input, object invalid

409

a customer already exists with the same account number
Up 
patch /transaction/deposit
perform a deposit (transactionDepositPatch)
deposit an amount that increases the balance of a given customer

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body transaction (required)
Body Parameter — Account number and amount to be deposited

Responses

200

successful transaction

400

invalid input, object invalid

404

customer not found
Up 
get /transaction
list transaction history (transactionGet)
Consult the transaction history of each movement by Date

Query parameters

initialDate (required)
Query Parameter — initial date to filter, the time will be considered at the start of the day (00:00:00) format: date
finalDate (required)
Query Parameter — final date to filter, the time will be considered at the end of the day (23:59:59) format: date

Responses

200

returns transactions filtered by date

204

no transactions

400

invalid date param
Up 
patch /transaction/withdraw

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body transaction (required)
Body Parameter — Account number and amount to be withdrawn

Responses

200

successful transaction

400

invalid input, object invalid or insufficient balance

404

customer not found
